The `useArrowNavigation` hook provides keyboard navigation for interactive elements using arrow keys. It supports row, column, and grid layouts with customizable behavior.

**Used in components:** [DropdownWrapper](/once-ui/components/dropdown-wrapper), [EmojiPicker](/once-ui/components/emoji-picker), [Select](/once-ui/components/select), [ContextMenu](/once-ui/components/context-menu), [MegaMenu](/once-ui/components/mega-menu)

<Feedback marginTop="16" marginBottom="24" description="Most of the time you don't need to use this hook directly. Components like DropdownWrapper, Select, and MegaMenu already provide built-in keyboard navigation. Use this hook when building custom navigable interfaces that aren't covered by existing components."/>

## Features

- Arrow key navigation (up, down, left, right)
- Support for row, column, and grid layouts
- Automatic focus management
- Enter/Space key selection
- Optional wrapping at boundaries
- Visual focus highlighting
- ARIA-compliant accessibility

## Basic usage

Add arrow navigation to a list:

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codeHeight={32}
  lineNumbers
  codes={[
    {
      code:
`import { useRef } from 'react';
import { useArrowNavigation } from '@once-ui-system/core';

function NavigableList({ items }: { items: string[] }) {
  const containerRef = useRef(null);
  
  const { handleKeyDown, focusedIndex } = useArrowNavigation({
    layout: 'column',
    itemCount: items.length,
    containerRef,
    onSelect: (index) => console.log('Selected:', items[index])
  });

  return (
    <Column ref={containerRef} onKeyDown={handleKeyDown}>
      {items.map((item, index) => (
        <Button
          key={index}
          data-focused={focusedIndex === index}
        >
          {item}
        </Button>
      ))}
    </Column>
  );
}`,
      language: "tsx",
      label: "NavigableList.tsx"
    }
  ]}
/>

## Grid layout

Navigate a grid with arrow keys:

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codeHeight={36}
  lineNumbers
  codes={[
    {
      code:
`import { useRef } from 'react';
import { useArrowNavigation } from '@once-ui-system/core';

function EmojiGrid({ emojis }: { emojis: string[] }) {
  const containerRef = useRef(null);
  
  const { handleKeyDown, focusedIndex } = useArrowNavigation({
    layout: 'grid',
    itemCount: emojis.length,
    columns: 8,
    containerRef,
    onSelect: (index) => selectEmoji(emojis[index]),
    wrap: true
  });

  return (
    <Grid
      ref={containerRef}
      onKeyDown={handleKeyDown}
      role="grid"
      columns={8}
    >
      {emojis.map((emoji, index) => (
        <Button
          key={index}
          role="gridcell"
          data-focused={focusedIndex === index}
        >
          {emoji}
        </Button>
      ))}
    </Grid>
  );
}`,
      language: "tsx",
      label: "EmojiGrid.tsx"
    }
  ]}
/>

## Row layout

Navigate horizontally:

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codeHeight={28}
  lineNumbers
  codes={[
    {
      code:
`function TabNavigation({ tabs }: { tabs: Tab[] }) {
  const containerRef = useRef(null);
  
  const { handleKeyDown, focusedIndex } = useArrowNavigation({
    layout: 'row',
    itemCount: tabs.length,
    containerRef,
    onSelect: (index) => setActiveTab(tabs[index].id),
    wrap: true
  });

  return (
    <Row ref={containerRef} onKeyDown={handleKeyDown} role="tablist">
      {tabs.map((tab, index) => (
        <Button
          key={tab.id}
          role="tab"
          aria-selected={focusedIndex === index}
        >
          {tab.label}
        </Button>
      ))}
    </Row>
  );
}`,
      language: "tsx",
      label: "TabNavigation.tsx"
    }
  ]}
/>

## Keyboard controls

- **Arrow Up**: Move focus up (column/grid)
- **Arrow Down**: Move focus down (column/grid)
- **Arrow Left**: Move focus left (row/grid)
- **Arrow Right**: Move focus right (row/grid)
- **Enter/Space**: Select focused item
- **Home**: Focus first item
- **End**: Focus last item

## Styling focused items

Use the `focusedIndex` to style the focused item:

<CodeBlock
  marginTop="16"

  lineNumbers
  codes={[
    {
      code:
`function StyledList() {
  const { handleKeyDown, focusedIndex } = useArrowNavigation({
    layout: 'column',
    itemCount: items.length
  });

  return (
    <div onKeyDown={handleKeyDown}>
      {items.map((item, index) => (
        <button
          key={index}
          className={focusedIndex === index ? 'focused' : ''}
          style={{
            outline: focusedIndex === index ? '2px solid blue' : 'none',
            transform: focusedIndex === index ? 'scale(1.05)' : 'scale(1)'
          }}
        >
          {item}
        </button>
      ))}
    </div>
  );
}`,

  ]}

## Accessibility

The hook automatically manages:

- Focus visibility with visual indicators
- ARIA roles and attributes
- Keyboard event handling
- Focus trap within container

For best accessibility, add appropriate ARIA attributes:

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codeHeight={20}
  lineNumbers
  codes={[
    {
      code:
`<div
  ref={containerRef}

  role="listbox"
  aria-label="Options"
>
  {items.map((item, index) => (
    <div
      key={index}
      role="option"
      aria-selected={focusedIndex === index}
      tabIndex={focusedIndex === index ? 0 : -1}
    >
      {item}
    </div>
  ))}
</div>`,
