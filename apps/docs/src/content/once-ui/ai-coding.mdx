## About

Once UI is an agent-first design system that empowers indie creators to build high-quality, responsive apps — without wrestling with complexity. Designed for AI-native workflows, it turns abstract intent into concrete interfaces with minimal syntax.

## Agent harness (recommended for codegen)

For **writing UI code**, use the harness — not full doc pages. It keeps context under ~10KB per task.

| Artifact | URL | When to load |
|----------|-----|--------------|
| Manifest | [/ai/manifest.json](/ai/manifest.json) | Entry point — lists all harness files |
| Compact rules | [/ai/rules.compact.md](/ai/rules.compact.md) | Always, before any UI task (~2KB) |
| Component catalog | [/ai/catalog.json](/ai/catalog.json) | Pick components by purpose/tag |
| Task bundle | [/ai/tasks/{intent}.json](/ai/tasks/index.json) | Match user intent (pricing, dashboard, …) |
| Component slice | [/ai/components/{Name}.json](/ai/catalog.json) | On demand, only components you use |
| Gold example | [/ai/examples/](/ai/tasks/index.json) | Mimic structure for the task type |
| Gotchas | [/ai/gotchas.json](/ai/gotchas.json) | When using Background, Card, RevealFx, Fade |

**npm imports** (version-pinned):

```bash
npm install @once-ui-system/core
```

```tsx
// Conceptual — read in your agent environment
// @once-ui-system/core/ai/rules.compact.md
// @once-ui-system/core/ai/tasks/pricing.json
// @once-ui-system/core/ai/components/Grid.json
```

**Workflow:**

1. Load `rules.compact.md`
2. Resolve intent → `tasks/{intent}.json`
3. Fetch component slices listed in the task bundle
4. Read the gold example for structure
5. Generate code
6. Validate: `pnpm validate-ai-code path/to/file.tsx`

## MCP Server (exploratory docs)

For **how does X work?** questions, agents can pull docs through [Context7's MCP server](https://context7.com/once-ui-system/core).

To configure it:

1. Open `mcp_config.json` in File / Preferences / Settings / Plugins

2. Add the following configuration:

```json
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp@latest"]
    }
  }
}
```

3. Restart your IDE
4. Run: `get-library-docs /once-ui-system/core`

Use Context7 for exploration. Use the harness for codegen.

## Syntax advantage

Compared to Tailwind + shadcn, Once UI yields up to **70% more compact** output, with **cleaner, more natural syntax** — semantic `Row`/`Column`, token props, and purpose-built components.

That compactness only pays off when agents load the harness and follow the rules — especially layout shorthands (`fill`, `center`), surface recipes, and component appropriateness (`Card` is interactive-only).
