## About

Once UI is an open-source design system built for indie creators, small teams, and startups to fuel the next generation of web applications. Developed and maintained by [Lorant](https://lorant.one) and the [Design Engineers Club](https://designengineers.club).

## License

Once UI is licensed under the MIT license.

## Quick start

Create a new Once UI app with our starter repository. It will install dependencies and set up the recommended structure:

<CodeBlock marginBottom="16"
codes={[
  {
    code: "npx create-once-ui-app@latest my-app",
    language: "bash",
    label: "CLI"
  },
  {
    code: "git clone https://github.com/once-ui-system/nextjs-starter.git",
    language: "bash",
    label: "Manual"
  }
]} />

Or add Once UI to an existing project:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "npm install @once-ui-system/core",
    language: "bash"
  }
]} />
