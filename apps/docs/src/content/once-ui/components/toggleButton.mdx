`ToggleButton` provides a flexible button-like element that can act as a toggle, tab, or segmented control. It supports selected state, prefix/suffix icons, and optional link behavior.

  marginTop="16"
  marginBottom="24"

      <ToggleButton label="Default" />
      <ToggleButton label="Link" href="#" />
    </Row>

`<ToggleButton selected label="Selected" />
<ToggleButton label="Default" />
<ToggleButton label="Link" href="#" />`,

      label: "Basic usage"
    }
  ]}
/>
