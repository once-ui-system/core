The `ScrollContainer` component creates a horizontal scrolling area with navigation buttons for browsing through collections of items. It features smooth scrolling, scroll snapping, and customizable control placement.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <ScrollContainer
      aspectRatio="16/9"
      items={Array(8).fill(0).map((_, i) => (
        <Column key={i} fill padding="8">
          <Background
            fill
            background="neutral-alpha-weak"
            border="neutral-alpha-medium"
            radius="l"
            lines={{ display: true, color: "neutral-alpha-weak" }}
          />
        </Column>
      ))}
    />
  }
  codes={[
    {
      code:
`<ScrollContainer
    items={[
      <YourContent />,
      <YourContent />,
      <YourContent />
    ]}
/>`,
      language: "tsx",
      label: "ScrollContainer"
    }
  ]}
/>

## Control Placement

Position navigation controls at the top or bottom with different horizontal alignments.

  marginBottom="24"
