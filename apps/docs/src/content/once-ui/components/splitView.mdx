The `SplitView` component creates a resizable divider between two panels, allowing users to adjust the size ratio. It supports both horizontal and vertical layouts with customizable resize handles.

## Usage Guidelines

- **Accessibility**: The resize handle is keyboard accessible and provides proper cursor feedback
- **Performance**: Uses efficient event handling with proper cleanup
- **Responsive**: Works well with responsive layouts and breakpoints
- **Constraints**: Set min/max split ratios to prevent panels from becoming too small
- **Visual Feedback**: Handle changes color when being dragged for better UX
