A container component that groups multiple Accordion components together with consistent styling and dividers between each item. It's useful for creating FAQ sections, navigation menus, and other grouped collapsible content.

AccordionGroup applies smart border-radius: the first item gets top-radius, the last item gets bottom-radius, and middle items have no radius — creating a cohesive grouped appearance.
