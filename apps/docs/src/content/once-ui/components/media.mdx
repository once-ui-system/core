`Media` handles responsive media content with support for lazy loading, aspect ratios, and intelligent rendering of videos or embeds. Use it for dynamic content blocks with consistent behavior.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Media
      priority
      sizes="(max-width: 768px) 100vw, 768px"
      src="/images/docs/once-ui/vibe-coding-light.jpg"
      alt="Preview"
      radius="xl"
      border="neutral-alpha-medium"
    />
  }
  codes={[
    {
      code:
`<Media
    src="/image.jpg"
    alt="Preview"
    sizes="(max-width: 768px) 100vw, 768px"
    radius="xl"
    border="neutral-alpha-medium"
/>`,
      language: "tsx",
      label: "Media"
    }
  ]}
/>

## Loading

Show a skeleton block while content is loading. Only works when `aspectRatio` is specified.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Media
      loading
      radius="l"
      aspectRatio="16/9"
      sizes="(max-width: 768px) 100vw, 768px"
      src="/images/docs/once-ui/vibe-coding-light.jpg"
    />
  }
  codes={[
    {
      code:
`<Media
    loading
    radius="l"
    aspectRatio="16/9"
    src="/image.jpg"
/>`,
      language: "tsx",
      label: "Skeleton"
    }
  ]}
/>

## Caption

The `caption` prop renders a `ReactNode` below the image, making it easy to provide additional details. It will render the image as `figure` and the caption as `figcaption`.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column fillWidth>
      <Media
        caption="Vibe code with Quasar! The time is now."
        src="/images/docs/once-ui/vibe-coding-light.jpg"
        sizes="(max-width: 768px) 100vw, 768px"
        alt="Preview"
        radius="xl"
        border="neutral-alpha-medium"
      />
    </Column>
  }
  codes={[
    {
      code:
`<Media
    caption="Vibe code with Quasar! The time is now."
    src="/image.jpg"
    alt="Preview"
    radius="xl"
    border="neutral-alpha-medium"
/>`,
      language: "tsx",
      label: "Caption"
    }
  ]}
/>

## Aspect Ratio

The `Media` component will automatically detect the aspect ratio of the image and set it to `auto` if not specified. You can provide an explicit aspect ratio using the `aspectRatio` prop.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  previewPadding="24"
  preview={
    <Media
      maxWidth={24}
      radius="l"
      aspectRatio="3/4"
      src="/images/docs/once-ui/vibe-coding-light.jpg"
    />
  }
  codes={[
    {
      code:
`<Media
    maxWidth={24}
    aspectRatio="3/4"
    radius="l"
    src="/image.jpg"
/>`,
      language: "tsx",
      label: "Aspect Ratio"
    }
  ]}
/>

## YouTube

The component auto-detects and renders YouTube links, just pass a YouTube URL to the `src` prop.

  marginBottom="24"

      radius="xl"
    />
