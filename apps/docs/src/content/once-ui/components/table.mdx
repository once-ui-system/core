`Table` is used to organize and render lists of information in a column/row format. It supports sorting, optional click handlers, and flexible cell content using `ReactNode`.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Table
      data={{
        headers: [
          { content: "Name", key: "name", sortable: true },
          { content: "Role", key: "role", sortable: true },
        ],
        rows: [
          ["Alice", "Engineer"],
          ["Bob", "Designer"],
          ["Carol", "Product"],
        ],
      }}
    />
  }
  codes={[
    {
      code:
`<Table
    data={{
      headers: [
        { content: "Name", key: "name" },
        { content: "Role", key: "role" },
      ],
      rows: [
        ["Alice", "Engineer"],
        ["Bob", "Designer"],
        ["Carol", "Product"],
      ],
    }}
/>`,
      language: "tsx",
      label: "Table"
    }
  ]}
/>

## Sorting

Enable sorting per column by setting `sortable: true` on the corresponding header. Click toggles between ascending and descending.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Table
      data={{
        headers: [
          { content: "Name", key: "name", sortable: true },
          { content: "Role", key: "role", sortable: true },
        ],
        rows: [
          ["Alice", "Engineer"],
          ["Bob", "Designer"],
          ["Carol", "Product"],
        ],
      }}
    />
  }
  codes={[
    {
      code:
`<Table
    data={{
      headers: [
        { content: "Name", key: "name", sortable: true },
        { content: "Role", key: "role", sortable: true },
      ],
      rows: [
        ["Alice", "Engineer"],
        ["Bob", "Designer"],
        ["Carol", "Product"],
      ],
    }}
/>`,
      language: "tsx",
      label: "Sorting"
    }
  ]}
/>

## Row actions

Add `onRowClick` to make rows interactive. A hover state is applied automatically.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codes={[
    {
      code:
`<Table
    onRowClick={(index) => alert(\`Clicked row \${index}\`)}
    data={{
      headers: [
        { content: "Task", key: "task" },
        { content: "Status", key: "status" },
      ],
      rows: [
        ["Launch", "In progress"],
        ["Prototype", "Done"],
      ],
    }}
/>`,
      language: "tsx",
      label: "Row actions"
    }
  ]}
/>

## Search

Enable `searchable` to filter rows by cell content. Use `searchPlaceholder` to customize the input label.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Table
      searchable
      searchPlaceholder="Filter by name..."
      data={{
        headers: [
          { content: "Name", key: "name", sortable: true },
          { content: "Role", key: "role" },
        ],
        rows: [
          ["Alice", "Engineer"],
          ["Bob", "Designer"],
          ["Carol", "Product"],
          ["Dan", "Marketing"],
        ],
      }}
    />
  }
  codes={[
    {
      code:
`<Table
    searchable

## Pagination

Set `paginated` to split large datasets into pages. Control the default page size with `defaultPageSize` and `pageSizeOptions`.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Table
      paginated
      defaultPageSize={3}
      data={{
        headers: [
          { content: "Name", key: "name" },
          { content: "Role", key: "role" },
        ],
        rows: [
          ["Alice", "Engineer"],
          ["Bob", "Designer"],
          ["Carol", "Product"],
          ["Dan", "Marketing"],
          ["Eve", "Support"],
          ["Frank", "Sales"],
        ],
      }}
    />
  }
  codes={[
    {
      code:
`<Table
  paginated
  defaultPageSize={5}
  pageSizeOptions={[5, 10, 20, 50]}
  data={{ headers, rows }}
/>`,
      language: "tsx",
      label: "Pagination"
    }
  ]}
/>

## Loading and empty states

Use `loading` to show a spinner while data is fetched. Provide `emptyState` or `loadingState` for custom placeholders.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Table
      loading
      data={{
        headers: [
          { content: "Name", key: "name" },
          { content: "Role", key: "role" },
        ],
        rows: [],
      }}
    />
  }
  codes={[
    {
      code:
`<Table
  loading
  loadingState={<Text>Fetching records...</Text>}
  emptyState={<Text>No results found.</Text>}
  data={{ headers, rows }}
/>`,
      language: "tsx",
      label: "Loading"
    }
  ]}
/>
