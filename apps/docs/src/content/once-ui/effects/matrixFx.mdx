The `MatrixFx` component creates an animated dot matrix reveal effect using the native Canvas 2D API. It supports instant reveal, progressive reveal on mount, or hover-triggered animations with smooth transitions. Perfect for hero sections, backgrounds, or interactive hover cards.

      trigger="mount"

## Bulge

Add dynamic wave effects to create fluid, organic motion. Choose between `ripple` (circular waves from center) or `wave` (S-curve flowing diagonally). Tiles are displaced, scale up at the wave front, and fade at the edges.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  previewPadding="0"
  preview={
    <MatrixFx
      height={24}
      colors={["accent-solid-strong"]}
      bulge={{
        type: "ripple",
        duration: 4,
        intensity: 15,
        repeat: true
      }}
    />
  }
  codes={[
    {
      code:
`<MatrixFx
    height={24}
    colors={["accent-solid-strong"]}
    bulge={{
      type: "ripple",   // Circular wave from center
      duration: 4,      // Wave duration in seconds
      intensity: 15,    // Displacement strength
      repeat: true      // Continuous cycles
    }}
/>`,
      language: "tsx",
      label: "Ripple"
    }
  ]}
/>

The `wave` type creates an organic, flowing S-curve that travels diagonally from bottom-left to top-right. The curve rotates and morphs as it moves, straightening at the edges for a natural liquid motion.

Use `trigger="manual"` with the `active` prop to control the effect from parent state.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  previewPadding="0"
  preview={
    <MatrixFxExample />
  }
  codes={[
    {
      code:
`"use client";

import React from "react";
import { MatrixFx, Row, ToggleButton } from "@once-ui-system/core";

export function Demo() {
    const [on, setOn] = React.useState(false);
    return (
      <Row gap="8" fillWidth>
        <ToggleButton onClick={() => setOn((v) => !v)}>
          {on ? "Stop" : "Start"}
        </ToggleButton>
        <MatrixFx height={16} trigger="manual" active={on} colors={["brand-solid-medium"]} />
      </Row>
    );
}
`,
      language: "tsx",
      label: "Manual Trigger"
    }
  ]}
/>
